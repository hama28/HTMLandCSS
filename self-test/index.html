Like

好きなものはボードゲーム

Dixit

『一枚の絵は、千の言葉に値する！』。このゲームのキャッチコピー。子供から大人まで誰でも楽しめて、答えがひとつじゃないゲーム。見ているだけでも楽しい。

ブラックストーリーズ

シチュエーションパズル。水平思考推理ゲームとか、yes/noパズルとも言われる。有名な問いは「ウミガメのスープ」。頭を柔らかくして考えるのが好きな人や、思考が閃く瞬間が好きな人にオススメ。Eureka!

ツンダール

シンプル。簡単なルールで、すぐに遊べる。運の要素も強いので、経験に関係なく遊べる。持ち運びやすい点もグッド。

BGA

ボードゲームアリーナ。オンラインでボードゲームが遊べる神サービス。StayHomeの影響でサーバがパンクするほど海外では人気。英語ができなくても楽しめるが、できるとより楽しめる。

Now

現在勉強中のスキル

React

触りだして1日。フロントエンドの実装によく使われるので勉強中。フロントで処理して動的に動かせる点が面白い。SPAの開発をやってみたい。

Ruby on Rails

触りだして2週間くらい。PythonやPHPと悩んだけど、実際に触ってみて一番書きやすかったRubyを選択。フレームワークが便利すぎて、フレームワークなしでは生きられない身体に。

SQLite

Railsの勉強してたらついてきたデータベース管理システム。軽量で扱いやすいらしいが、他を知らないので利便性を感じたことはない。フロントエンドがある程度できるようになってからMySQLの勉強を始める予定。

GitHub

ソースコード管理。最近、リモートリポジトリの設定を学び、コードを共有してどこでも開発できるようになった。最初はGitの設定に苦戦したが、今となってはGit無しとかありえん身体に。複数人での開発の経験もしてみたい。

Mirai

これから習得したいスキル

クラウド

AWSやGCPといったクラウドインフラ。今や常識になっているので、クラウドサーバでの開発を経験したい。フロントエンドがある程度できるようになってから、AWSなどを使ってサービスのリリースを目指す。

Docker

コンテナ仮想化でよく聞くやつ。今のところさっぱり分からん。説明を聞いて便利なのは分かるが、体験してみないとなんとも…。クラウドでも使えるみたいなので、AWSなどと一緒に使ってみる。

人工知能

機械学習。最近よく見るLINEのチャットボットとかを作ってみたい。特に、自然言語処理が気になる。機械学習はPythonが強く、Rubyはライブラリが少ないのが残念…。Pythonの勉強もしたい。

Firebase

超高速でアプリの開発ができるプラットフォーム。調べれば調べるほど面白そうで、早く触ってみたい。Amazonが提供しているAmplifyも気になる。
